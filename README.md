
# Mars-rover TP-FINAL

## Installation et commande

Pour installer le programme

```bash
  npm install
```
Pour lancer le rover
```bash
  npm run start:rover
```
Pour lancer le controlleur du rover
```bash
  npm run start:mission
```
Pour lancer les test 
```bash
  npm test
```
